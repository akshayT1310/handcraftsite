# Logo Setup Instructions

## To add your logo to the website:

1. **Save your logo image** as `logo.png` in this directory:
   ```
   d:\handcraft\handcraftsite\static\images\logo.png
   ```

2. The logo is already configured to display in the navigation bar with:
   - The logo image on the left
   - "HANDCRAFT" and "ARTISAN CREATIONS" text beside it

3. **Recommended logo specifications:**
   - Format: PNG with transparent background
   - Size: Approximately 200x200 pixels (will be scaled to 60px height)
   - Aspect ratio: Square or slightly wider works best
